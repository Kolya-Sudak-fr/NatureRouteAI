# NatureRoute AI

AI powered travel route planner for nature lovers.

The app helps users generate travel routes based on their nature preferences such as mountains, forests, lakes, and national parks.

## Tech Stack

Swift  
SwiftUI  
MVVM (planned)  
MapKit (planned)

## Features (planned)

• Nature preference selection  
• AI generated travel routes  
• Interactive map with routes  
• Offline route storage  
• Travel route customization
